在网上看到这么一个解决方案

  1. 确保自己电脑系统时间准确；
  2. 打开 Chrome 浏览器；
  3. 访问「微信网页版」，并扫描登录；
  4. 在左侧找到该联系人，选中后对话，右侧会显示进入聊天窗口；
  5. 把你要发的内容写在输入框内，恩，不要点「发送（Send）」；
  6. 打开调试台，方法为：工具（Tools）- JavaScript 控制台（JavaScript Console）；
  7. 在控制台输入 new Date().toLocaleString()，回车，查看输出的信息；因为不同电脑的输出可能不同，所以请了解自己系统的日期格式后再继续；比如我这里的输出为：6/3/2013 5:58:00 PM
  8. 想清楚自己要定时的时间，按照上述格式写好，如：6/3/2013 8:00:00 PM

该方法的不足

  • 你定时发，对方不一定同时收到，可能有网络延时；
  • 登录 Web 微信后，手机端会收不到推送，不过不影响手机的微信正常聊天；
  • 该方法只能适用于一次性发送，就是说设定一个时间，发送一个指定内容；
  • 噢，只要你会 JavaScript 编程，完全可以实现每一个小时准点发一段不同的话，诸如这样的逻辑，并不难；
