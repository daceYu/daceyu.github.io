原生JS：String对象

String()是一个用于创建字符串对象的构造器

下面我们创建一个字符串对象和一个基本字符串： 

var s_obj = new String('prtatoes');
typeof s_obj;       // output => object
var s_prim = 'patatoes';
typeof s_prim;      // output => string

s_obj === s_prim;   // output => false
s_obj == s_prim;    // output => true

String()构造器的成员

String.fromCharCode(code1, code2, code3,...)

该方法会根据用户输入的字符编码来创建字符串，并将其返回 

String.fromCharCode(115, 99, 114, 105, 112, 116);   // 'script'

String.prototype的成员

length

该属性返回字符串的字符数量

charAt(position) && charCodeAt(position)

charAt()返回指定位置处的字符，位置从0开始计数
charCodeAt()返回指定位置处字符的unicode编码 

'script'.charAt(0);         // output => 's'
'script'.charCodeAt(0);     // output => 115
concat(str1, str2, str3,...)

将各个字符串连接成一个新的字符串 

"".concat("hello ","world");    // output => hello world
indexOf(needle, start) && lastIndexOf(needle, start)

该方法用于返回匹配串的起始位置，第二个参数是可选的，用于指定搜索的开始位置。如果没有找到匹配串，就返回-1

indexOf()是从前往后搜索，lastIndexOf()是从后往前搜索。 

let str = "javascript";
str.indexOf("a");       // output => 1
str.indexOf("a", 5);    // output => -1
str.lastIndexOf("a");   // output => 3
localeCompare(needle)

该方法会将两个字符串放在当前区域内进行比对，如果两个字符串完全相同就返回0，如果needle中的字符串被包含，就返回1，否则返回-1 

let str = "script";
str.localeCompare("script");    // output => 0
str.localeCompare("cript");     // output => 1
str.localeCompare("sscript");   // output => -1
match(regexp)

该方法会根据其接受的正则表达式对象返回一个容纳所有匹配串的数组 

let str = "R2-D2 and C-3";
str.match(/[0-9]/g);     // ["2", "2", "3"]
replace(needle, replacement)

该方法用于替换字符串对象中所有匹配相关正则表达式的内容。另外，我们还可以通过replacement参数设置一系列不同的匹配模式，例如$1、$2...$9查看详情 

let str = "R2-D2";
str.replace(/2/g, '-two');   // "R-two-D-two"
str.replace(/2/g, '$1$1');   // "R22-D22"
search(regexp)

该方法会返回正则表达式匹配的第一个字符串的位置 

"C-3PO".search(/[0-9]/)     // 2
slice(start, end)
