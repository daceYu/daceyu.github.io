原生JS：String对象

String()是一个用于创建字符串对象的构造器

下面我们创建一个字符串对象和一个基本字符串： 

var s_obj = new String('prtatoes');
typeof s_obj;       // output => object
var s_prim = 'patatoes';
typeof s_prim;      // output => string

s_obj === s_prim;   // output => false
s_obj == s_prim;    // output => true

String()构造器的成员

String.fromCharCode(code1, code2, code3,...)

该方法会根据用户输入的字符编码来创建字符串，并将其返回 

String.fromCharCode(115, 99, 114, 105, 112, 116);   // 'script'

String.prototype的成员

length

该属性返回字符串的字符数量

charAt(position) && charCodeAt(position)

charAt()返回指定位置处的字符，位置从0开始计数
charCodeAt()返回指定位置处字符的unicode编码 

'script'.charAt(0);         // output => 's'
'script'.charCodeAt(0);     // output => 115
concat(str1, str2, str3,...)

将各个字符串连接成一个新的字符串 

"".concat("hello ","world");    // output => hello world
indexOf(needle, start) && lastIndexOf(needle, start)

该方法用于返回匹配串的起始位置，第二个参数是可选的，用于指定搜索的开始位置。如果没有找到匹配串，就返回-1

indexOf()是从前往后搜索，lastIndexOf()是从后往前搜索。 

let str = "javascript";
str.indexOf("a");       // output => 1
str.indexOf("a", 5);    // output => -1
str.lastIndexOf("a");   // output => 3
localeCompare(needle)

该方法会将两个字符串放在当前区域内进行比对，如果两个字符串完全相同就返回0，如果needle中的字符串被包含，就返回1，否则返回-1 

let str = "script";
str.localeCompare("script");    // output => 0
str.localeCompare("cript");     // output => 1
str.localeCompare("sscript");   // output => -1
match(regexp)

该方法会根据其接受的正则表达式对象返回一个容纳所有匹配串的数组 

let str = "R2-D2 and C-3";
str.match(/[0-9]/g);     // ["2", "2", "3"]
replace(needle, replacement)

该方法用于替换字符串对象中所有匹配相关正则表达式的内容。另外，我们还可以通过replacement参数设置一系列不同的匹配模式，例如$1、$2...$9查看详情 

let str = "R2-D2";
str.replace(/2/g, '-two');   // "R-two-D-two"
str.replace(/2/g, '$1$1');   // "R22-D22"
search(regexp)

该方法会返回正则表达式匹配的第一个字符串的位置 

"C-3PO".search(/[0-9]/)     // 2
slice(start, end)

该方法会返回字符串中start到end之间的部分。如果start的 值是个负数，那么开始位置实际上等于length + start，同样的， 如果end的值是负数，其结束位置等于length + end 

let str = "hello, I am daceyu";
str.slice(2,8);     // output => "llo, I"
str.slice(2,-2);    // output => "llo, I am dace"
substring(start, end)

该方法与slice()基本相同，如果start或end为负值、无效值时，会被视为0处理 ，如果它们的值大于length，则会被视为length，如果start的值大于end，则它们会自动交换彼此的值。 

let str = "hello, I am daceyu";
str.substring(2,8);     // output => "llo, I"
str.substring(8,2);     // output => "llo, I"
split(separator, limit)

该方法可以将字符串转成一个数组。它的第二个参数limit是可选的，separator参数可以是一个正则表达式，也可以是字符串。 

let str = "1,2,3,4";
str.split(/,/);     // output => ["1", "2", "3", "4"]
str.split(",", 3);  // output => ["1", "2", "3"]
toLowerCase() && toLocaleLowerCase()

这两种方法都是将字符串转为小写 

"JAVA".toLowerCase();    // output => "java"
toUpperCase() && toLocaleUpperCase()

这两种方法都是将字符串转为大写 

"javaScript".toUpperCase();     // output => "JAVASCRIPT"

ES5的补充

String.prototype.trim()

该方法用来去除字符串首尾两端的空字符。 

var str = " fskafa ";
str.trim();    // output => "fskafa"

ES3中可以这么实现： 

str.replace(/(^\s*)|(\s*$)/g, "");

ES6中新增的方法

includes(str, position)

这个方法用来查找字符串中是否存在某一个值，如果存在返回true，如果不存在返回false。参数str为需要查询字符(串)，参数position为查询的起始位置 

let str="wbiokr";
str.includes('wb');     // output => true
str.includes('wbiokr'); // output => true
str.includes('w',1);    // output => false
str.includes('b',1);    // output => true
startsWith(str, position) && endsWith(str, position)

参数str为需要查询字符(串)，参数position为查询的起始位置，返回布尔值。

startsWith()表示是否位于字符的头部位置。
endsWith()表示是否位于字符的尾部位置。 

let str="wbiokr";
str.startsWith('wb');   // output => true
str.startsWith('w');    // output => true
str.startsWith('w',1);  // output => false
str.startsWith('b',1);  // output => true

str.endsWith('kr');     // output => true
str.endsWith('r');      // output => true
str.endsWith('i',6);    // output => false
str.endsWith('i',3);    // output => true
str.endsWith('kr',6);   // output => true
repeat(num)

string.repeat(num)方法,参数num为重复字符串的次数，各种情况下的返回值见代码 

let str='hi';
str.repeat(3);        // output => 'hihihi' str被重复了3次返回，str不改变
str.repeat(2.6);      // output => 'hihi'   浮点类型参数，会取整重复
str.repeat(0);        // output => ''       0的时候返回空字符串
str.repeat(undefined);// output => ''       undefined转化为0
str.repeat(null);     // output => ''       null转化为0
str.repeat(NaN);      // output => ''       NaN转化为0
str.repeat(-3);       // Uncaught RangeError:Invalid count value
for...of

字符串相关遍历方法 

let str = "wbiokr";
for (let i of str) {
    console.log(i);
    if (i === "w") return;
}
`${}`

过去我们都是通过字符串拼接进行添加变量。`${}`可以直接拼接。 

let str = "wbiokr";
$('#box').html(`there is a word ${str},i know it`);
// there is a word wbiokr,i know it
